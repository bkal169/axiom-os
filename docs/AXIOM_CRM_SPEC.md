# AXIOM CRM: Intelligent Capital Markets & Real Assets Platform

## MISSION
Build an AI-native CRM that unifies institutional capital markets intelligence with 
field agent productivity, eliminating friction across the entire lifecycle: 
lead generation → capital formation → asset management → exit.

## CORE PHILOSOPHY
Axiom is designed around a fundamental insight: investors and agents need the SAME 
data but through DIFFERENT lenses. Investors need portfolio intelligence and compliance. 
Agents need lead velocity and conversion tools. Axiom serves both without compromise.
